Si bien la la variable X tiene probabilidad cero para todo valor fuera del recorrido, esta función de distribución es

Si 0 ≤t < 1 ⇒FX(t) = P(X ≤t) = P(X = 0) = 0,0019 (el único punto de probabilidad positiva es X = 0).
Si 1 ≤t < 2 ⇒FX(t) = P(X ≤t) = P(X = 0) + P(X = 1) = 0,1981 (los únicos valores del recorrido
menores o iguales que t son X = 0 y X = 1).
Si t ≥2 ⇒FX(t) = P(X ≤t) = P(X = 0) + P(X = 1) + P(X = 2) = 1 (se acumulan todos los valores del
recorrido).

Item a

saltos en los valores del recorrido. Entonces, viendo FX(t), podemos deducir que RX(t) = {0, 1, 2, 3}.
